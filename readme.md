# 🤖 AI Products & Apps Catalog

A dynamic, searchable catalog of AI tools and applications, categorized by use cases.

## Features
- 🔍 Search by keyword, category, or use case
- 📁 Browse by categories (Video Generation, Text Generation, etc.)
- ⭐ Ratings and pricing information
- 🔗 Direct links to tools
